What is Python?

Python is a programming language popular with beginners. It is a high-level language meaning that it is written in a form that is similar to our human language.

Python was created by Guido van Rossum and was released in 1991

What can Python do?

Python can be used to make games. Some examples of games built using Python are Battlefield 2, Bridge Commander, World of War Tanks etc.

Python can be used to make applications. Some examples of applications built using Python are Instagram, Spotify, Netflix, Dropbox etc.

Python can be used for data manipulations, AI and machine learning, web development etc.

Why should you use Python?

Python is the easiest programming language to learn. This is due to the simple python syntax allowing the code to be easily readable and debuggable.

Python can be used on many platforms like Windows, Mac, Linux etc.

Python Syntax

Python syntax allows users to write code with fewer lines compared to other programming languages.

Python uses new lines to complete a command, unlike other programming languages which use parenthesis or semicolons.

Python uses indentation and whitespaces to show a function, iterations and classes (these topics will be covered in the More Python section).

Python Data Types

From the previous section, you may wonder why printing hello world uses quotation marks but printing a number doesn’t. This is because Python has different data types.


x and y are different in this situation although they both store 9, x stores the integer 9 (the value) but y stores the string version of 9.

Strings (str) - Strings are arrays of bytes representing Unicode characters ( Unicode - Universal encoding standard use for different programming languages, each letter, digits or symbol assigned to a unique numerical value). It is enclosed by double or single quotation marks.

Integers (int) - Integers are negative or positive whole number ( doesn’t include fractions) and they are not enclosed by quotation marks.

Floats (float) - Floats are real numbers represented with a floating-point represented by decimal points

Booleans (bool) - Booleans are True or False values. True and False with capital (T and F) are valid booleans otherwise Python will give an error
